# Changelog

All notable changes to the Rosary iOS app will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-03-03

### Added
- Complete traditional Rosary (three sets of mysteries: Joyful, Sorrowful, Glorious)
- Novena support: 54-day Rosary, Sacred Heart, Miraculous Medal, St. Joseph, St. Jude, St. Therese, Perpetual Help
- Redesigned rosary bead UI with larger 3D beads and rosary-shaped layout
- App icon and launch screen
- GitHub Actions CI workflow for iOS builds
- Pre-push hook auto-install script
- Privacy policy and App Store compliance notes
- LICENSE and security gitignore patterns

### Fixed
- Removed Luminous Mysteries (traditional Rosary has three sets only)
- Corrected traditional mystery day assignments per pre-2002 custom
- Domain updated from ecclesia.dev to ecclesiadev.com in PRIVACY.md
- Registered NovenaListView, Novena, NovenaLibrary, NovenaTracker in xcodeproj

### Accessibility
- Full accessibility audit and fixes

### Theology
- Mystery day assignments corrected per Bellarmine review
